Allocate markers and marking workflow

Allocate markers in Assignment settings allows markers to be assigned to grade specific students.

Assignment settings

Enable Marker allocation by turning on Marking workflow

  1. Ensure editing is on.
  2. Click Add an activity or resource.
  3. Choose Assignment, click Add.
  4. Choose your base assignment settings.
    See Working with assignments for more information.
  5. Scroll down to the section Grade
    • Select Yes for Use marking workflow*
    • Select Yes for Use marking allocation
    • Once complete, click Save and return to unit.

* Use marking workflow must be selected to use marking allocation.

Allocating individual markers

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissionslink.
  3. Scroll down to the Optionssection.
  4. If Quick grading is off, select Quick grading to turn it on, scroll back to the grading table.

    Status = workflow, Marker = marking allocation
  5. For each student select the desired marker from the Marker dropdown.
  6. Once complete, click

Please note: once the Status has reached Marking completed the Marker cannot be modified.

Allocate markers in bulk

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissions link.
  3. Ensure Quick grading is on.
  4. Select the students to be allocated to one marker.
  5. Scroll to the bottom of the grading table, in the drop down with selected..., select Set allocated marker
  6. Click , then click OK to confirm the action.
  7. From the drop down Allocated Marker select the marker.
  8. Click Save changes

The marker's name will now appear in the submission information.

Managing marking Workflow

Marking workflow allows a lecturer to control the information available to students on the marking status of submitted work, to review and moderate results and to release feedback and grades to students once grading is completed.

Changing workflow status for individual assignments

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissions link.
  3. Ensure Quick grading is on.
  4. For each student select the desire workflow state

    Workflow states
    • Not marked - initial state
    • In marking - marking has begun, typically after the due date
    • Marking completed - submission has been marked but marker able to check and revise
    • In review - submission(s) are being reviewed (often part of a moderation phase)
    • Ready for release - review complete
    • Released - feedback and grade released to the students
  5. Once complete, click

Changing workflow status in bulk

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissions link.
  3. Ensure Quick grading is on.
  4. Select the students to have their workflow state changed.
  5. Scroll to the bottom of the grading table, in the drop down labelled with selected..., select Set marking workflow state
  6. Click , click OK to confirm the action.
  7. The selected users will be listed, from the drop down Marking workflow stateselect the status.
  8. Click Save changes

Allocating individual markers

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissions link.

  3. The Separate groups drop down (when viewing all) shows the user's groups at the top followed by others. If user is in only one group this is the default selection.
  4. Choose the group from the drop down menu.
  5. The students in the selected group are displayed.



  6. Scroll down to the Options section. Turn Quick grading ON. Scroll back to the grading table.
     
    Status = workflow, Marker = marking allocation
  7. For each student select the desired marker from the Marker dropdown.
  8. Once complete, click

Allocate markers in bulk

  1. On the unit home page, click on the Assignment link that you want to grade.
  2. On the assignment page, click on the View/Grade all submissions link.
     
  3. Click on Separate groups to select which participants you would like to mark. You can select All participants or participants in a certain group. 

  4. Ensure Quick grading is on.
  5. Check the box next to the individual students to be allocated to one marker or check the Select box to allocate all students in the group in a single step.
  6. Scroll to the bottom of the grading table, in the drop down with selected..., select Set allocated marker


  7. Click   , then click OK to confirm the action.
  8. From the drop down Allocated Marker select the marker.
  9. Click Save changes
